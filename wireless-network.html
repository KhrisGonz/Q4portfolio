Wireless Network Description

Wireless networks allow devices to connect and communicate using radio waves instead of physical cables. Common types include Wi-Fi, Bluetooth, and cellular networks.

Hardware Needed

To set up a wireless network, the following hardware is typically used:

  • Wireless Router
  • Wireless Access Points (APs)
  • Wireless Network Interface Cards (Wi-Fi adapters)
  • Antennas (optional, for range boosting)

Pros & Cons

Pros

  • Greater mobility and flexibility
  • Easy and fast setup
  • Scalable and accessible for many devices

Cons

  • Susceptible to interference
  • Lower speed compared to wired networks
  • Less secure without proper encryption
